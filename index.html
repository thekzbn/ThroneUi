ThroneUi

Ultra-Minimalist Design System

A vanilla HTML/CSS/JavaScript UI library built on strict constraints: 3 colors maximum, no shadows, minimal borders, typography-first design.

View Components Documentation

Design Philosophy

3 Colors Only

Gold (#d6ad60), Black (#000), White (#fff). No exceptions, no gradients, no additional colors.

No Shadows

Clean, flat design aesthetic. Visual hierarchy through typography, spacing, and color contrast only.

Minimal Borders

Borders only where absolutely necessary: forms, profile images, and certification cards.
